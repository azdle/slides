# Using this Site Yourself

You can use this as a "host" yourself by setting the `href` query parameter to a
url of a markdown file using the format:

    # Slide 1
    This is a slide.
    Note: Speaker Note
    ---
    # Slide 2a
    This is a new slide. 
    ^^^
    #Slide 2b
    This is a vertical slide.
